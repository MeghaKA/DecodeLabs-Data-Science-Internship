# Fraud Detection Pipeline using Machine Learning

## Project Overview

This project focuses on building a supervised machine learning pipeline for detecting suspicious transactions using classification algorithms.

The main objective is to handle imbalanced transaction data and evaluate models using appropriate fraud detection metrics instead of relying only on accuracy.

## Objectives

- Perform exploratory data analysis
- Handle missing values and preprocessing
- Analyse class imbalance
- Apply SMOTE for minority class balancing
- Train classification models
- Compare Logistic Regression and Random Forest performance
- Evaluate using Precision, Recall and ROC-AUC

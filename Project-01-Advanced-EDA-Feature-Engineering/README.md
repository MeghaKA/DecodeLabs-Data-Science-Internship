# E-Commerce Sales Data Analysis

# Project Overview

This project analyzes an e-commerce sales dataset containing customer transactions, product purchases, payment methods, referral sources, and order statuses.

The objective is to uncover business insights through exploratory data analysis, data cleaning, feature engineering, and visualization techniques.



# Dataset

The dataset contains:

* 1,200 transactions
* Customer information
* Product purchases
* Payment methods
* Order status information
* Referral sources
* Revenue metrics



# Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn



# Project Workflow

Phase 1: Exploratory Data Analysis

* Dataset inspection
* Missing value analysis
* Outlier detection
* Distribution analysis
* Correlation analysis

Phase 2: Data Cleaning

* Missing value handling
* Date conversion
* Feature engineering

Phase 3: Business Analysis

* Product revenue analysis
* Payment method analysis
* Referral source analysis
* Order status analysis
* Monthly revenue trends



# Key Findings

* CouponCode contained 25.75% missing values.
* TotalPrice showed several high-value outliers.
* Product sales contributed unevenly to total revenue.
* Payment methods influenced revenue patterns.
* Revenue demonstrated seasonal variation across months.
